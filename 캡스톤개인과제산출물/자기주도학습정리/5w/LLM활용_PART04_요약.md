## Chapter 10. GPT 탐색: 나만의 GPT 활용

### 1. 맞춤형 GPT 개요

- 사용자의 목적에 따라 GPT의 **성격, 스타일, 전문 영역**을 자유롭게 설정
- **별도 프로그래밍 지식 불필요** — 자연어 지침만으로 생성 가능
- ChatGPT **Pro(유료)** 사용자 대상 기능

**설정 가능한 요소:**

| 요소 | 내용 |
|------|------|
| 주제 범위 조정 | AI의 지식 범위 확장 |
| 응답 길이 조정 | 사용자 요구에 맞춘 출력 길이 |
| 대화 톤 조정 | 사용자 선호에 맞는 말투 |

**활용 예시:** 학생용 학습 튜터 GPT, 직장용 업무 보조 GPT, 고객 상담 챗봇 등

---

### 2. '내 GPT' 만들기 기초

**진입 경로:**
- 프로필 아이콘 클릭 → **'내 GPT'** 선택 → **'+ GPT 만들기'** 클릭
- 또는 GPT 탐색 화면 우측 상단 **'+ 만들기'**

**화면 구성:**

```
[왼쪽: 만들기 / 구성 탭]   [오른쪽: 미리보기]
  ↓ 역할과 목적 입력          ↓ 실시간 GPT 테스트
```

---

### 3. '내 GPT' 구성 요소 상세

#### 3-1. 만들기 탭 — 역할과 목적 작성

"어떤 GPT를 만들고 싶으신가요?"에 자연어로 입력. 예시:

> `"당신은 대학생들을 위한 PowerPoint 발표 자료를 만드는 데 특화된 전문 컴퓨터 과학 튜터입니다"`

#### 3-2. 구성 탭 — 세부 설정

| 항목 | 내용 |
|------|------|
| 이름 | GPT의 표시 이름 |
| 설명 | GPT의 역할 요약 |
| 지침 (Instructions) | 응답 형식, 대상 수준, 출력 구조 등을 상세 명시 |
| 대화 스타터 | 사용자에게 보여줄 예시 질문 버튼 |
| 지식 (Knowledge) | 참조 파일(PDF, 문서 등) 업로드 |
| 기능 | 웹 검색, 캔버스, DALL-E 이미지 생성, 코드 인터프리터, 데이터 분석 |
| 작업 (Actions) | 외부 API 연동 자동화 |

---

### 4. 실습 예시: SlidePro for Computer Science

**목적:** 컴퓨터 과학 전공 학생 대상 기술적 PowerPoint 콘텐츠 생성

**지침 핵심 내용:**

```
슬라이드 구조:
1. 슬라이드 제목
2. 소주제 3-5개
3. 각 소주제 아래 설명 문단 1-2개 (200-300자)
4. Python 코드 블록
5. 코드 설명 (줄 단위)

- 대상 수준: 2~3학년 컴공 학생
- 슬라이드 수 미지정 시 기본값 10개
- 시각 자료(다이어그램) 선택적 제안
```

**대화 스타터 예시:**
- "Make 10 detailed slides explaining convolutional neural networks."
- "Create a presentation on Dijkstra's algorithm with Python code."

---

### 5. '내 GPT' 수정 및 공유

#### 지침 수정 포인트
- 설명 구문을 명사형으로 통일
- 답변 마지막에 **파일 저장 옵션** 포함:
  ```
  - [ ] PDF
  - [ ] PowerPoint (PPTX)
  - [ ] Jupyter Notebook (ipynb)
  ```

#### 한글 PDF 지원 이슈 및 해결
- ChatGPT 기본 상태에서 한글 PDF 생성 시 글꼴 깨짐 발생
- 해결: 지침에 한글 폰트 사용 지시 추가 + **나눔고딕 등 한글 폰트 파일을 '지식'에 업로드**

#### 공유 방법
1. 우측 상단 **[만들기]** 버튼 클릭
2. 공유 범위 선택: GPT 스토어 공개 / 초대된 사람만 / 나만 보기
3. 이메일로 특정 사용자 초대 가능
4. **[업데이트]** 버튼으로 저장 완료

#### 프로필 이미지 등록
- 왼쪽 상단 **'+'** 버튼 → 이미지 업로드 → GPT 대표 아이콘 설정

### 7. 실습 예시 2: 공공데이터 분석 GPT

**역할 정의:**
> `"너는 공공데이터 기반 데이터 분석 전문가야. 사용자는 보통 통계청, 공공데이터포털, 행안부 등에서 받은 데이터를 업로드하거나 붙여넣을 수 있어."`

**지침 핵심:**
- 열(column) 구조 요약 및 통계 분석
- pandas, matplotlib, seaborn 기반 코드 예시 제공
- 분석 결과를 초보자도 이해할 수 있게 단계별 설명
- 공공정책·행정·교육·지역 격차 맥락을 고려한 인사이트 제시

**지식 파일:** `koreanize_matplotlib` 라이브러리 업로드 → 한글 시각화 지원

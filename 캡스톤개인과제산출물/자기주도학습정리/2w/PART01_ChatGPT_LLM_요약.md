## 1. 생성형 AI와 거대 언어 모델(LLM) 개요

### 1.1 생성형 AI (Generative AI)

기존 데이터를 학습하여 **새로운 콘텐츠(텍스트·이미지·음악·비디오)를 생성**하는 AI 기술.  
2016년 AlphaGo 이후 AI가 비약적으로 발전하면서 2018년부터 ChatGPT, Gemini, Claude 등 LLM 서비스가 등장했다.

| 특징 | 설명 |
|------|------|
| 콘텐츠 생성 | 입력에 따라 텍스트·이미지·음악 등 새로운 결과물을 생성 |
| 창의성 | 학습 데이터를 기반으로 새로운 아이디어·콘텐츠를 만들어내는 능력 |
| 응용 분야 | 게임·영화·광고·교육 등 다양한 산업에서 맞춤형 콘텐츠 제공 |

### 1.2 거대 언어 모델 (LLM, Large Language Model)

NLP 분야에서 활용되는 생성형 AI의 특수 유형.  
방대한 텍스트 데이터로 **사전학습(Pre-training)** 하여 문맥 이해 및 자연어 생성에 특화.

| 특징 | 설명 |
|------|------|
| 언어 이해 | 문맥 파악 + 문법적으로 올바른 문장 생성. 번역·요약·QA 등에 활용 |
| 대화 생성 | 자연스럽고 일관된 대화 생성. 챗봇·개인비서 서비스에 적합 |
| 지식 기반 | 학습 데이터 기반으로 다양한 주제에 대한 정보 제공 |

## 3. OpenAI & GPT 발전 역사

### 3.1 OpenAI 개요

| 항목 | 내용 |
|------|------|
| 설립 | 2015년 — Elon Musk, Sam Altman, Reid Hoffman 등 공동 창립 |
| 목표 | AI 기술 민주화 — 강력한 AI가 소수 기업/정부에 독점되는 것을 방지 |
| 구조 변화 | 비영리 연구소 → 2019년 영리 자회사(OpenAI LP) 설립, MS 투자 유치 |
| 핵심 프로젝트 | GPT 시리즈: "대량 텍스트 데이터를 사전학습한 생성형 Transformer" |

### 3.2 GPT/ChatGPT 버전별 발전

| 버전 | 출시 | 매개변수 | 주요 특징 |
|------|------|----------|-----------|
| GPT-1.0 | 2018 | 1.17억 | Transformer 기반 최초 GPT. 사전훈련+미세조정 패러다임 제시 |
| GPT-2.0 | 2019 | 15억 | 텍스트 생성 능력 대폭 향상. 악용 우려로 초기 비공개 출시 |
| GPT-3.0 | 2020 | 1,750억 | 본격적 실용 LLM. 다양한 NLP 태스크에서 SOTA 달성 |
| ChatGPT-1.0 | 2022 | ~1,750억 | GPT-3.5 기반 대화형 챗봇. **RLHF** 도입으로 정렬(alignment) 강화 |
| ChatGPT-2.0 | 2023 | - | GPT-4 기반. 논리·창의적 사고 강화 |
| GPT-4o | 2024 | - | 멀티모달(텍스트+이미지+오디오) 통합 처리 |
| o3 / o3-mini | 2024~25 | - | Chain-of-Thought 추론 강화. STEM 분야 성능 대폭 향상 |
| ChatGPT-4.5 | 2025 | - | 환각현상 감소, 자연스러운 상호작용 개선 |

## 4. 핵심 기술 용어

| 용어 | 설명 |
|------|------|
| **GPT** | Generative Pre-trained Transformer. 대량 텍스트를 사전학습한 뒤 다양한 언어 태스크를 수행하는 Transformer 기반 모델 |
| **사전훈련 (Pre-training)** | 인터넷·책 등 방대한 텍스트 데이터로 단어/문장 관계, 문법, 지식을 학습하는 단계 |
| **미세조정 (Fine-tuning)** | 사전훈련 완료 후 특정 도메인 데이터를 추가 학습하여 특화 성능 향상 |
| **RLHF** | Reinforcement Learning from Human Feedback. 인간 피드백으로 모델 출력을 강화학습하여 정렬(alignment) 개선. ChatGPT 1.0에서 핵심 기법으로 도입 |
| **멀티모달** | 텍스트뿐 아니라 이미지·오디오·비디오 등 다양한 형태의 입력을 동시에 처리하는 능력 |
| **환각현상 (Hallucination)** | LLM이 사실과 다른 내용을 자신감 있게 생성하는 현상. 신뢰성의 핵심 이슈 |
| **프롬프트** | 사용자가 LLM에 입력하는 질문·명령어. 품질이 출력 결과에 직접 영향 |

## 5. ChatGPT 활용 가이드

### 5.1 접속 및 기본 사용

| 항목 | 내용 |
|------|------|
| 접속 URL | `chatgpt.com` — 회원가입 없이도 기본 질의 가능. 가입 시 대화 기록·개인화 기능 사용 |
| 회원가입 | 이메일 또는 Google/Microsoft/Apple 계정으로 가입 후 이메일·전화 인증 |
| 요금제 | Free(GPT-4o 기본) / Plus $20/월(고급 기능) / Pro $200/월(무제한) |
| 대화창 | 중앙 채팅창에 질문 입력 → 즉각 답변. 질문 수정 재전송, 공유 링크 생성 가능 |

### 5.2 맞춤 설정 (Personalization)

`[메뉴 → ChatGPT 맞춤 설정]`에서 사용자 정보·응답 스타일·특성을 지정하면 개인 비서처럼 활용 가능.

| 설정 항목 | 내용 |
|-----------|------|
| 자기소개 입력 | 직업, 관심사, 사용 목적 등을 입력하면 맥락에 맞는 답변 제공 |
| 응답 스타일 | 어조(반말/존댓말), 답변 길이, 단계별 제시 형식 등을 지정 |
| ChatGPT 특성 | "Let's think step by step" 등 CoT 유도 프롬프트 삽입 가능 |
| 적용 범위 | "새 채팅에 사용" 체크 후 저장 → 이후 새 대화부터 설정 반영 |
